Installation

  1. Open the Extensions sidebar in VS Code
  2. Search for Joker and choose "Joker Theme" by Leandro Nascimento
  3. Click Install
  4. Click Reload to reload VS Code
  5. Navigate to File > Preferences > Color Theme and select Joker Theme
  6. You're all set! Enjoy! ðŸŽ‰

Best Font for theme

The font used in the previews is Fira Code, available on Fira Code Guithub.

After you've downloaded and installed it, open your editor's settings and add the following...

"editor.fontFamily": "Fira Code Medium"
